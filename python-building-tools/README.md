# python-building-tools

Command-line and subprocess examples, plus several `slidecount` variants.

## Directory map

- `arguments.py`: inspect CLI argument behavior.
- `sub.py`, `sub-freespace.py`: subprocess command examples.
- `slidecount*.py`: different argument-parsing approaches for counting slides in `.pptx` files.

## Notes

- These scripts are tutorial-style and intentionally show multiple approaches to similar tasks.
- Prefer `slidecount-argparse.py` for modern CLI usage.
- Basic smoke tests live in `tests/`.
