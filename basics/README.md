# basics

Small Python practice scripts for core language features and standard library usage.

## What is here

- `hello.py`, `helloworld.py`, `printBlah.py`: simple entrypoint/printing examples.
- `functions.py`, `lists.py`, `dict.py`, `tuples.py`, `sets.py`, `strings.py`, `loops.py`: basic data structure and control-flow demos.
- `error_handling.py`: try/except usage example.
- `threads.py`: small threading demo.
- `os.py`: runtime/system information and safe directory operations.
- `planets.py`: custom exception and validation pattern.
- `convert_image.py`: OpenCV-based line-art conversion utility.

## Notes

- Scripts are now structured with import-safe functions and `if __name__ == "__main__"` guards.
- Unit tests live in `tests/test_basics.py`.
