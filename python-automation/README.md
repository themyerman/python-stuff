# python-automation

Automation examples for filesystem tasks, scraping, and Selenium browser interaction.

## Directory map

- `OrganizeMe/`: file categorization constants and organizer examples.
- `Source Code/`: automation scripts:
  - local automation (`organizeDirectory.py`, `batchingCommands.py`, `fileIO.py`)
  - web scraping (`scrape.py`, `scrapePages.py`)
  - Selenium demos (`webBrowsing.py`, `interactions.py`, `dragAndDrop.py`)

## Notes

- Scripts are educational samples; prefer running them in isolated test directories.
- Selenium scripts require a local browser driver and network access.
- Tests for deterministic utilities live in `tests/`.
