# python-essential-training

Chapter-based training examples covering core Python concepts.

## Directory map

- `Chap01`..`Chap15`: progressive, tutorial-style scripts by topic.
- `Chap15/jurl/`: a larger example app with CGI/database helper modules.

## Notes

- This folder is intentionally educational and includes older coding patterns for teaching progression.
- To avoid changing lesson intent, optimization refactors should be selective and done chapter-by-chapter.
- Recommended next step: migrate one chapter at a time to modern style with dedicated tests per chapter.
