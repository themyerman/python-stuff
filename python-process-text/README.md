# python-process-text

Text-processing and NLP examples using NLTK, sklearn, and simple file I/O.

## Directory map

- `code/open_spark.py`: reads source text data.
- `code/use-tokens.py`, `code/use-ngrams.py`, `code/use_pos.py`, `code/use-nltk.py`: tokenization and NLP examples.
- `code/use_idftdf.py`: TF-IDF vectorization example.

## Notes

- Several scripts download or rely on external corpora/packages at runtime.
- For reproducibility, install dependencies first and run in a controlled environment.
- Basic script smoke tests live in `tests/`.
